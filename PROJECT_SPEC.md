Project: `react-native-boilerplate`

Purpose: The intetion is to use this as scaffolding to help setup new react native projects quickly and easily

New apps can ask the LLM to initialize a new project with this hello world

And we could iterate and improve the boilerplate repo over time so that new apps get the latest guidelines, conventsion, linting, packages, etc.
